Để deploy OpenClaw thì thường sẽ sử dụng một dịch vụ VPS (Virtual Private Server) để chạy nó, trực tiếp hoặc sử dụng Docker để chạy.

<h1>Setup OpenClaw trong môi trường Docker</h1>

https://www.youtube.com/watch?v=-aeR1cQktdM

Trong ví dụ này, em sẽ deploy OpenClaw trong Docker trên một VM chạy Ubuntu (chung VM với những ví dụ trước)

**Setup Docker trong VM**

Chạy script:
```
curl -fsSL https://get.docker.com -o get-docker.sh
sh get-docker.sh
```

Test đã cài đặt được chưa:
```
sudo docker run hello-world
```

Note:

Xem danh sách images/containers: docker image/container ls

Xóa mọi container: docker rm $(docker ps –a –q)

docker container prune: xóa tất cả container đang không hoạt động

docker stop <tên container/container ID>: dừng container

docker stop $(docker ps –a –q): dừng mọi container

docker rmi <image ID>: xóa image

**Tạo image OpenClaw cho Docker**

Cần thực hiện git clone về repo source code của OpenClaw và chạy script setup cho docker.

```
git clone https://github.com/openclaw/openclaw.git 
cd openclaw
mkdir -p ~/.openclaw/workspace
chown -R 1000:1000 ~/.openclaw
./docker-setup.sh
```
